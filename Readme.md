## GitStudy

* 查看版本
	* git version
* 查看git配置
	* git config --list
* 初始化仓库
	* git init
* 查看状态
	* git status
	* git status -s 简化格式
* 查看修改信息
	* git diff
* 添加修改
	* git add 文件名
* 提交
	* git commit -m "注释信息"
* 查看日志版本
	* git log
	* git log --pretty=oneline 一行显示
	
* 回退到指定版本
	* git reset --hard HEAD 版本
* 回退到上个版本
	* git reset --hard HEAD^
* 退两个版本
	* git reset --hard HEAD~2
* 查询操作记录
	* git reflog (--pretty=oneline)简写版本号
* 直接回退到一个指定的版本
	* git reset --hard 版本号
	
* 查看当前所在分支/查看所有分支
	* git branch
* 在当前版本上创建新的分支
	* git branch 分支名
* 切换分支
	* git checkout 分支名

* 合并分支A
	* git merge A
